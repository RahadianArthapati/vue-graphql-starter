# vue-graphql-starter

### I use [yarn](https://yarnpkg.com) for package manager
![graphcool][gc_img]
1. create simple API with [**graphcool**][gc]
2. type **_yarn_** in root project
3. run the dev server with <b>yarn run start</b>
